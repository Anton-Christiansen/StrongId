StrongId - A strongly typed identifier

Following the spirit of Domain-Driven-Design and value objects, this makes it easy to source generate strongly typed identifiers for your entities.
Mark your classes with an [StrongId] Attribute. The attribute has 2 optional parameters: A 'ParameterType' (default to System.Guidd) and 'ParameterName' (defaults to Id)

Available as a nuget package:
                
    Anton_Christiansen.StrongId

For seemingly integration with Entity Framework, you can add the nuget package:

    Anton_Christiansen.StrongId.EntityFramework

Example of use

Key.cs:
```c#
    using Daemon.Models.Shared;
    using StrongId.Attributes;
    
    namespace Daemon.Models.Api;
    
    [StrongId]
    public partial class Key
    {
        // Ef core
        private Key(KeyId id, Timestamp created)
        {
            Id = id;
            Created = created;
        }
        
        public Tag Tag { get; set; } = null!;
        public Private Private { get; set; } = null!;
        public Website? Website { get; set; }
        public Description? Description { get; set; }
        public Timestamp Created { get; init; }
        
    
        public static Key Create(Tag tag, Private @private, Website? website = null, Description? description = null)
        {
            return new Key(KeyId.Create(Guid.NewGuid()), Timestamp.Now)
            {
                Tag = tag,
                Private = @private,
                Website = website,
                Description = description
            };
        }
    }
```

Then in your DbContext you will override ConfigureConvections

DatabaseContext.cs:
```c#
    protected override void ConfigureConventions(ModelConfigurationBuilder configurationBuilder)
    {
        configurationBuilder.AddStrongIdConverters();
    }
```

Example of generated code:

From the package `Anton_Christiansen.StrongId`

Key.g.cs:
```c#
    // <auto-generated/>

    using System;
    
    namespace Daemon.Models.Api;
    
    partial class Key
    {
    	public KeyId Id { get; private set; }
    }
```

KeyId.g.cs:
```c#
    // <auto-generated/>

    using System;
    
    namespace Daemon.Models.Api;
    
    public readonly record struct KeyId
    {
    	public required System.Guid Value { get; init; }
    
    	public static KeyId Create(System.Guid id) => new KeyId
    	{
    		Value = id
    	};
    }
```

From the package `Anton_Christiansen.StrongId.EntityFramework`

KeyIdConverter.g.cs
```c#
    // <auto-generated/>

    using Daemon.Models.Api;
    using Microsoft.EntityFrameworkCore.Storage.ValueConversion;
    
    namespace StrongId.EntityFramework.Converters;
    
    internal sealed class KeyIdConverter() : ValueConverter<KeyId, System.Guid>
    (x => x.Value, x => KeyId.Create(x));
```

StrongIdEntityFrameworkExtensions.g.cs
```c#
    // <auto-generated/>
    using Microsoft.EntityFrameworkCore;
    using StrongId.EntityFramework.Converters;
    using Daemon.Models.Api;
    
    namespace StrongId.EntityFramework.Extensions;
    
    internal static class StrongIdEntityFrameworkExtensions
    {
    	extension(ModelConfigurationBuilder configurationBuilder)
    	{
    		internal void AddStrongIdConverters()
    		{
    			configurationBuilder.Properties<KeyId>()
    			  .HaveConversion<KeyIdConverter>();
    
    		}
    	}
    }
```
